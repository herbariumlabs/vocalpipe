**Guidelines for implementation:**

This is a special scheme initiated during 2014-15 aimed at all-round development of floriculture sector in the State which incorporates cultivation, marketing, training etc. For the year 2015- 16 cultivation of Gerbera under Shade net house with tissue cultured planting materials is proposed to be taken up covering an area of 200 sqm. per unit area.

| 1\. |  | Govt. will provide planting materials, required quantity of shade net and one garden tool kit as Govt. share |  |
| ----- | :---- | :---- | :---- |
| 2. |  | The beneficiaries will have to erect the bamboo/steel	structure themselves under observationof departmental officersas beneficiary's share |  |
| 3. |  | The cost of bed preparation, fertilizer etc. will have to be borne by the beneficiaries |  |
| 4\. |  | Departmental officers will provide complete technical support to the beneficiaries |  |
|  **Eligibilitycriteria for selectionof beneficiary:** |  |  |  |
| 1\. |  | The component will be implemented in rural and urban areas |  |
| 2. |  | The beneficiariesshould be of small and marginal category |  |
| 3. |  | Women SHGs and unemployed youths should get preference |  |
| 4. |  | The beneficiaries should have their own land in their premises or rented land on long term lease |  |
| 5. |  | Modern cultivation technology will disseminated by the concerning departmental officers through training and skill teaching |  |
| 6\. |  | One beneficiary can avail only one unit @200 sqm. |  |
| 7\. |  | Proper representation from SC and ST farmers should be ensured |  |
| 8\. |  | Beneficiary selection will be finalized by the District Level Beneficiaries Selection Committee through **wide**publicity in mass media |  |
| 9\. |  | The District Level Beneficiaries Selection Committee will monitor the programme time to time and DAO/ONO will report the progress of work to HQ every fortnight. |  |
